# Adonis Webpack

Create Hello World Adonis Project with Webpack Integration.

## Installation

Requirements
```bash
npm install -g adonis-cli
npm install -g nodemon
```

Setup dependencies in local project
```bash
# create new project
adonis new adonis-webpack
cd adonis-webpack
# install dependecies
npm install
```

## Usage

First run

```bash
npm run serve:dev
```
